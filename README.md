# DragonflyPdf

Dragonfly analyser and processor for PDFs.

IMPORTANT: Requires [ImageMagick](http://imagemagick.org)

Get:
    aspect_ratio
    number_of_pages

Make:
    page_thumb

## Installation

Add this line to your application's Gemfile:

    gem 'dragonfly_pdf'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dragonfly_pdf

## Usage

Analyser returns
    aspect_ratio
    number_of_pages

Processor makes
    page_thumb
